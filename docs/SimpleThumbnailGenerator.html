getEmitter() → {Object}

Get the event emitter. The first argument to the listener is the event type. Events: - thumbnailsChanged after a thumbnail is added or removed. - newThumbnail when there is a new thumbnail. The second argument is the thumbnail. - thumbnailRemoved when a thumbnail is removed. The second argument is the thumbnail. - playlistEnded when the playlist has ended and all thumbnails have been generated. - finished when the stream has been removed and all thumbnails have expired. - error if an exception is thrown before the generator has initialized.
